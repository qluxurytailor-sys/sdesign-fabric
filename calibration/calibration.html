S-DESIGN — Size Calibration / Mapping

โปรแกรมแยกสำหรับคำนวณไซส์/วาง layout จากค่าที่วัดผ่าน Illustrator

1 โหลด Profile JSON

วัดค่าจาก .ai ครั้งแรกผ่าน extract-sizes.jsx แล้วโหลดไฟล์ที่ได้มาตรงนี้ — ครั้งถัดไปไม่ต้องเปิด Illustrator

ลากไฟล์ .json มาวางที่นี่ หรือคลิกเพื่อเลือก
ยังไม่มี profile

2 เลือกไซส์ที่จะคำนวณ + กรอกชื่อนักกีฬา (ถ้ามี)

ติ๊กไซส์ที่อยากให้ระบบคำนวณ layout ให้ — ไซส์ฐาน (base) จะมีเส้นขอบสีส้ม

0 / 280 mm

3 ผลการคำนวณ + การตัดสินใจ (decisions)

แต่ละการ์ดคือไซส์หนึ่งใบ — preview แสดง layout, รายการด้านล่างคือสิ่งที่ระบบเสนอให้ทำตอนประกอบไฟล์ AI จริง

4 ส่งออก decisions JSON (ส่งต่อให้ Illustrator ประกอบไฟล์จริง)

ไฟล์ JSON นี้คือ "คำสั่ง" ที่บอก script Illustrator ว่าแต่ละไซส์ต้องขยาย/หมุน/ย้ายอะไรบ้าง — ใช้แทนการให้ Illustrator คิด layout เอง


    


    
    

      
? วิธีใช้

      

        
Workflow:

        
    
          
  1. เปิดไฟล์ .ai ของไซส์ทั้งหมดใน Illustrator → จัด layer/ชื่อชิ้นตามมาตรฐาน → File → Scripts → extract-sizes.jsx → ได้ไฟล์ *.profile.json
    2. 
          
  2. โหลด profile JSON ในขั้นที่ 1
    3. 
          
  3. เลือกไซส์ + กรอกชื่อในขั้นที่ 2 → กด "คำนวณ Layout"
    4. 
          
  4. ตรวจผลในขั้นที่ 3 ว่ามี collision หรือต้องหมุนแขน/ขยาย artboard ไหม
    5. 
          
  5. โหลด decisions.json ในขั้นที่ 4 → ส่งให้ script ประกอบไฟล์ AI ต่อไป (ไม่ต้องให้ Illustrator คิด layout เอง)
    6. 
        

        
มาตรฐานการตั้งชื่อใน Illustrator (เลือกอย่างใดอย่างหนึ่ง):

        
    
          
  • แบบ A: Layer ชื่อตามไซส์ S, M, L, XL, 2XL, 3XL, 4XL ภายในมีชิ้นชื่อ front, back, sleeveL, sleeveR, collar, stripe
    • 
          
  • แบบ B: ชื่อชิ้นตรงๆ M_front, XL_sleeveL ฯลฯ
    • 
        

        
หน่วย: ทุกค่าใน JSON เป็น mm โดย origin = มุมซ้ายบนของ artboard, Y เพิ่มลงล่าง
